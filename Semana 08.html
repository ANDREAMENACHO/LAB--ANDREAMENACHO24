Conocimos Livewave y Netlify

Introducción

En esta semana conocimos otras herramientas que nos ayudarìan a crear nuestro enlace web, que permitìa que el profesor pueda ver nuestra bitacora .

Clase Virtual

En esta clase, el profesor nos instruyó sobre ChatGPT y cómo esta herramienta podría facilitarnos la generación de código HTML de manera sencilla y práctica. Nos mostró ejemplos como solicitarle a ChatGPT un código HTML para "Hola Mundo". También nos introdujo a herramientas como LiveAwey y Netlify, que nos ayudarían a crear enlaces de nuestros repositorios de GitHub en la web.

Clase Virtual

Debido a las dificultades que enfrentaba al intentar vincular Visual Studio Code con GitHub, decidí registrarme en Netlify y crear un sitio donde pude subir toda mi carpeta de archivos. Con esto, generé un enlace propio para acceder a mi trabajo. Luego nos fuimos a salas, donde de manera grupal nos ayudábamos a resolver dudas y el profesor cada cierto tiempo pasa por cada uno de los grupos para revisar nuestros avances y resolver las dudas que más se repetía en el grupo.

Clase Presencial

Durante la clase presencial, el profesor nos introdujo al concepto de design thinking y nos enseñó cómo descargar e instalar tipografías desde Google Fonts, además de explicar cómo vincularlas en Visual Studio Code. Luego, nos animó a practicar desde las computadoras del aula para asegurarse de que todos entendieran el proceso o para resolver cualquier duda que surgiera.

Clase Presencial

Además, el profesor nos proporcionó pautas sobre los criterios que tendría en cuenta para la bitácora, y nos presentó las métricas que se utilizarán para evaluar cada etapa del proyecto final. También nos asignó una actividad grupal en la que teníamos que identificar un problema y realizar una lluvia de ideas sobre sus necesidades para comprenderlo mejor.
