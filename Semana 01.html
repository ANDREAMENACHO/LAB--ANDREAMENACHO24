Introducción

Este fue mi primer día clases, el curso se llevaba dos veces a la semana en modalidad semivirtual, el profesor nos dio algunos lineamientos acerca del proyecto que entregaríamos en la semana 05.

Clase Virtual

Esta primera clase el profesor nos enseño lo que era GitHub y visual studio code, programas que nos ayudarían a poder crear nuestra bitácora digital. Por lo que, tuvimos que crearnos una cuenta de GitHub y un repositorio en él, y descargarnos el software de visual studio code, ya que todos los cambios se realizarían desde aquí.

Clase Presencial

A su vez descargamos una plantilla que ayudaría a estructurar y embellecer nuestra bitácora, para ello tuvimos que crear una ruta llamada index.html dentro de github, en ella subimos todos nuestra plantilla, y en visual studio enlazamos una carpeta donde todos cambios o modificaciones se guardarían en el.
